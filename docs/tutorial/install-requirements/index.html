0. Install Requirements

Install Git and Node.js to get started with Doks.

On this page

  • Git
    • Check Git install
  • Node.js
    • Check Node.js install
    • Check npm install

Git#

Install the latest source release of Git.

Check Git install#

git --version

Node.js#

Install the latest LTS version or newer of Node.js.

Why Node.js? Doks uses npm (included with Node.js) to centralize dependency management, making it easy to update resources, build tooling, plugins, and build scripts.

Check Node.js install#

node --version

Check npm install#

npm --version
